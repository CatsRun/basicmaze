# Basic Maze - not finished

A simple maze that uses keybaord input to move. 

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. Load the program
2. Click: run python program
3. A new window will open with the game

Instructions for using the software:

1. Use the arrow keys to move your ball. 
2. Avoid the moving ball.
3. Find the exit
